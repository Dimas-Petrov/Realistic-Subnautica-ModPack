# Установка

## Инструкции

Для игры в модпак - Скачайте последнюю версию инсталлятора модпака, запустите от имени администратра и пройдите этапы инсталляции. 
Обратите внимание на страницу компонентов - там вы сможете выбрать:

    1. Необязательные для установки в игру моды. Это делается на ваше усмотрение.
    2. Выбрать озвучку игрового процесса.
    3. Выбрать пресет для решейда - один или несколько.

!!! warning "Предупреждение"
    - Учтите, что в Inno Setup (инструмент установки) я лично встроил код, который умеет автоматически обнаруживать где у вас установлена копия игры. Установщик не учитывает версию игры, то есть, есть ли у вас вторая её копия в памяти системы. Убедительная просьба проверять куда устанавливается игра. Пример пути установки: 

    "**С:\Games\Subnautica**", либо "**E:\SteamLibrary\steamapps\common\Subnautica**".

!!! note "Проблемы во время установки МодПака и их решение"
    Если при установке МодПака возникают ошибки нужно:

    1. Убедится в том, что вы запустили установку от имени администратора. 
    2. Удалить файл, из-за которого возникла ошибка, если инсталлятор пишет "файл уже существует". 
    3. Удалить МодПак предыдущей версии (в случае, если будет удаляться МодПак версии 0.4.0 будут удалены файлы сохранений игры, рекомендуется создать их резервную копию перед тем как удалять).
