# Realistic Subnautica ModPack — 2025 Edition

## 📘 О проекте
**Realistic Subnautica ModPack** — это сопровождаемый сообществом проект по моддингу игры **Subnautica**, направленный на улучшение реализма игрового процесса и расширение возможностей игры за счёт тщательно подобранных модификаций.

Основные цели проекта:
- усиление ощущения реализма;
- улучшение качества игрового процесса мелкими, но значимыми деталями;
- повышение стабильности и совместимости модов;
- предоставление удобного установщика с автоматическими проверками.

МодПак изначально создавался для версии игры **2023**, но начиная с версии **0.7.0**, сборка официально поддерживает **только Subnautica 2025 Edition**.

---

## ⚙ Требования

### 1. Системные требования
**Минимум:** соответствие системных требований самой игры  
**Рекомендуется:**
- SATA SSD или лучше  
- 8 ГБ ОЗУ  
- GTX 1060Ti / аналог AMD  
- CPU: AMD Phenom II X6 или лучше  

### 2. Лицензия
**Минимум:** иметь установленную копию игры Subnautica  
**Рекомендуется:** лицензионная копия Steam / GOG / Epic Games Store  

> Установщик проверяет наличие лицензии, но **не блокирует установку** на нелицензионные копии.  
> Пользователь самостоятельно несёт ответственность за соблюдение законодательства и условий использования игры.

### 3. Операционная система
**Минимум:** Windows 7 SP1 x64  
**Рекомендуется:** Windows 10/11 (последние обновления)  
**Альтернативно:** любой Linux‑дистрибутив с ядром **Linux 7.0+** (через Proton/Wine)

---

## 📥 Инструкция по установке

1. Скачать установщик (.exe) последней версии из удобного источника  
   *(рекомендуется GitHub Releases)*  
2. Запустить установщик  
3. Принять условия **EULA**  
4. Выбрать необходимые компоненты  
5. Проверить корректность пути установки  
6. Закрыть Subnautica перед установкой  
7. Нажать «Установить» и дождаться завершения процесса
